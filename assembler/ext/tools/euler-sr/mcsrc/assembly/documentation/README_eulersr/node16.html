Cleaning vector and masked sequences from SANGER reads.

The vector and masked sequences in Sanger reads can cause problems for error correction and assembly. These can be filtered by piping them through two utilities 

cat reads.fasta ${EUSRC}/assembly\_utils/RemoveMaskedSequence.pl |
${EUSRC}/assembly\_utils/RemoveVectorSequence.pl > reads.trimmed
