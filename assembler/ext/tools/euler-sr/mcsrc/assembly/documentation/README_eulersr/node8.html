CLEANING UP DATA

The assembly quality is strongly dependent on how clean the data is. Quality values may be used to trim/remove low quality reads. If you are assembling Illumina reads, particularly those produced by the first generation sequencer, some extra programs are provided to filter reads that have errors particular to the system (discussed below).
