Sample data.

Provided with the assembly is a sample paired end reads file (reads.fasta), a rule file for this (readtitles.rules), and a sample file that contains reads from two haplotypes (reads.variants.fasta).

To assemble the reads file: 

	${EUSRC}/assembly/Assemble.pl reads.fasta 25 readtitle.rules.
