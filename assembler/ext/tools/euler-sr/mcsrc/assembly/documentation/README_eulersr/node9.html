Translating data.

EULER-SR takes as input a single fasta file with ALL reads, an optional mate file that specifies mate-pairing (which may be generated), and a rule file that describes how to map a fasta title to a clone type.

Utilities are provided to translate from: .sff (the 454 binary output file). .fastq (Sanger format) .fastq (Illumina quality values).
