FASTQ -> Fasta (Illumina)

Illumina uses a different scaling of quality values than the normal phred scores. Although qualityTrimmer attempts to guess the type (Sanger vs. Illumina), this may be forced on the command line: 

	${EUSRC}/assembly/x86\_64/qualityTrimmer	-fasta reads.fastq -outFasta reads.fasta -type illumina
