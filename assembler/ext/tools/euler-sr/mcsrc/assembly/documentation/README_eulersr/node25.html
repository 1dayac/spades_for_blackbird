Detecting fragmented assemblies.

If you have mate-pairs, the assembly will be in the 'matetransformed' directory, otherwise the 'transformed' directory. For examples here the 'transformed' directory is used.

Print all 'dead-end' edges: 

		 ${EUSRC}/assembly/${MACHTYPE}/printGraphSummary transformed/reads.fasta -sources

This will print edge indices followed by their lengths. If there is a very high number of dead-end edges, the coverage is probably low causing a fragmented assembly. Some sequencing platforms such as Illumina have a lot of "coverage deserts", such as in GC-rich regions, where the coverage is very low, causing fragmentation of the assembly.
